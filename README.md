# Linux学習リポジトリ (Linux Learning Repository)

## 概要 (Overview)
このリポジトリは、**Linux標準教科書**で学んだ内容をまとめるために利用します。
各章ごとに学習内容、コマンド例、練習問題の解答などを記録していきます。

This repository is used to summarize the content learned from the **Linux Standard Textbook**.
Learning contents, command examples, and practice problem solutions are recorded for each chapter.

## 目的 (Purpose)
- Linux標準教科書の学習内容を体系的に整理する
- 学んだコマンドや概念を実践的に記録する
- 後から参照しやすい形で知識をまとめる

## ディレクトリ構成 (Directory Structure)
```
Linux-learning/
├── README.md                  # このファイル
├── chapters/                  # 各章の学習内容
│   ├── chapter01/            # 第1章
│   ├── chapter02/            # 第2章
│   └── ...
├── commands/                  # コマンドリファレンス
└── exercises/                 # 演習問題と解答
```

## 学習の進め方 (How to Study)
1. 各章のディレクトリに学習ノートを作成
2. 実際にコマンドを実行して結果を記録
3. 重要なポイントや疑問点をメモ
4. 演習問題に取り組み、解答を記録

## Linux標準教科書について (About Linux Standard Textbook)
Linux標準教科書は、LPI-Japanが提供する無料のLinux入門教材です。
Linuxの基本的な操作やコマンド、システム管理の基礎を学ぶことができます。

- 公式サイト: https://linuc.org/textbooks/linux/
