# コマンドリファレンス (Command Reference)

このディレクトリには、学習したLinuxコマンドのリファレンスをまとめます。

## 使い方 (Usage)

各コマンドについて、以下の情報を記録します：

- コマンド名と基本的な書式
- 主要なオプション
- 実用的な使用例
- 注意点やTips

## コマンド一覧 (Command List)

カテゴリ別にコマンドを整理します：

### ファイル・ディレクトリ操作
- `ls` - ファイル一覧表示
- `cd` - ディレクトリ移動
- `pwd` - 現在のディレクトリ表示
- `mkdir` - ディレクトリ作成
- `cp` - ファイルコピー
- `mv` - ファイル移動・名前変更
- `rm` - ファイル削除

### ファイル内容の表示・編集
- `cat` - ファイル内容表示
- `less` / `more` - ページャー
- `head` / `tail` - ファイルの先頭・末尾表示
- `vi` / `vim` - テキストエディタ

### システム情報
- `uname` - システム情報表示
- `whoami` - ユーザー名表示
- `date` - 日付・時刻表示
- `df` - ディスク使用量
- `free` - メモリ使用量

### プロセス管理
- `ps` - プロセス一覧
- `top` - プロセス監視
- `kill` - プロセス終了
