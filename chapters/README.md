# 章別学習ノート (Chapter Notes)

このディレクトリには、Linux標準教科書の各章ごとの学習内容をまとめます。

## 章の構成 (Chapter Structure)

各章のディレクトリには以下のファイルを含めます：

- `README.md` - 章の概要と学習目標
- `notes.md` - 詳細な学習ノート
- `commands.md` - その章で学んだコマンドのまとめ
- `examples/` - 実践例やスクリプト

## 学習の記録方法 (How to Record)

1. 各章のディレクトリを作成
2. その章の重要ポイントをメモ
3. コマンドを実際に実行して結果を記録
4. 理解できなかった点や疑問点も記録
