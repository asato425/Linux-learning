# 演習問題 (Exercises)

このディレクトリには、Linux標準教科書の演習問題とその解答をまとめます。

## 使い方 (Usage)

各章の演習問題について：

1. 問題文を記録
2. 自分で考えた解答を記録
3. 実際にコマンドを実行して結果を確認
4. 学んだポイントや反省点をメモ

## ディレクトリ構成 (Directory Structure)

```
exercises/
├── README.md           # このファイル
├── chapter01/          # 第1章の演習
├── chapter02/          # 第2章の演習
└── ...
```

各章のディレクトリには以下のファイルを含めます：

- `problems.md` - 演習問題
- `solutions.md` - 解答
- `notes.md` - 学んだポイント

## 演習の進め方 (How to Practice)

1. まず自分で問題を解いてみる
2. コマンドを実際に実行して動作を確認
3. わからない場合は教科書を見直す
4. 解答と自分の回答を比較
5. 理解を深めるためのメモを追加

## Tips

- コマンドの実行結果は具体的に記録する
- エラーが出た場合も、その原因と解決方法を記録
- 同じ目的を達成する別の方法があれば、それも記録
